# School Quest Tracker

A lightweight, fun tracker for school assignments and at-home practice. It includes:

- Subjects for Math, English, Humanities, Sciences, Flex, plus Field Trip and Special Project.
- Assignment tracking with due dates and progress.
- Points and streak system to reward consistent usage.
- Reward shop where parents can add point-based rewards.
- Quick links to home practice tools: IXL Math, IXL English, Je Lis Libre, Duolingo, and Math-Aids.

## Run locally

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
